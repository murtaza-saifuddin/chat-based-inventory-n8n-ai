# Chat-Based Inventory Management System (n8n + Agentic AI)

This project is a chat-based inventory management system built using n8n and Agentic AI.

## Features
- Update stock via chat commands
- Add new items automatically
- Read real-time inventory from Google Sheets
- AI Agent decides when to read or update data

## Tech Stack
- n8n
- Agentic AI (LLM)
- Google Sheets
- Workflow Automation

## How it works
1. User sends a chat message
2. AI Agent understands intent
3. Inventory is updated or read from Google Sheets

## Use Case
Small businesses can manage inventory using simple chat commands without manual spreadsheets.
